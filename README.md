# Railway Web Service

A simple Node.js web service ready for deployment on Railway.

## Deployment

1.  Create a new GitHub repository and push this code to it.
2.  Go to [railway.app](https://railway.app).
3.  Click "Login" and authorize with your GitHub account.
4.  Click "New Project".
5.  Select "Deploy from GitHub repo" and choose the repository you created.
6.  Railway will automatically detect the `start` script and deploy the service.

## Local Development

1.  Install dependencies: `npm install`
2.  Start the server: `npm start`
3.  The service will be available at `http://localhost:3000`.
